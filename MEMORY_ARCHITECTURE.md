## 音乐老友 记忆系统框架

### 分层架构

```
┌──────────────────────────────────────────────────────┐
│                    记 忆 分 层                        │
├─────────────┬──────────────┬─────────────────────────┤
│  工作记忆    │   情景记忆     │      长期偏好            │
│  L1: WM     │   L2: EM      │      L3: LTM            │
├─────────────┼──────────────┼─────────────────────────┤
│ 当前对话     │ 近期交互+状态  │  稳定用户画像             │
│ LLM窗口直接  │ DB + 衰减     │  持久化存储               │
│ 仅当前会话   │ 小时~周       │  永久                     │
└─────────────┴──────────────┴─────────────────────────┘
```

---

### 数据模型

```
db.json
├── messages[]        ← WM: 最近200条聊天（LLM上下文来源）
├── plays[]           ← EM: 最近500次播放（去重+模式分析源）
├── songStats{}       ← EM: 每首歌播放次数/最后时间（跨session去重）
├── artistStats{}     ← EM: 每位歌手播放次数（长期偏好统计）
├── session{}         ← 累计播放总数/总时长
└── prefs
    ├── facts[]       ← LTM: 结构化事实 [{id,category,content,confidence,count,created,updated}]
    ├── summary       ← LTM: 跨会话摘要（最近5个会话块）
    ├── patterns      ← LTM: 播放模式（时段+歌手）
    └── topArtists[]  ← 快照：最近偏好歌手

user/
└── memory.md         ← LTM: 冻结快照（Agent手写长期记忆，Human-editable）
```

---

### 事实（Facts）分类

| Category | 示例 | 衰减策略 |
|----------|------|---------|
| `mood` | "用户感到疲惫" | 指数衰减，半衰期3天 |
| `preference` | "喜欢听周杰伦" | 线性衰减，7天后每天-1% |
| `user_habit` | "用户深夜听歌" | 线性衰减 |
| `event` | "用户在备考" | 线性衰减 |
| `discovery` | "听过《晴天》" | 线性衰减 |
| `relationship` | "周杰伦→深夜" | 线性衰减 |

### 写-管-读 三轮循环

```
                    ┌──────────────┐
                    │   用  户     │
                    └──┬───────┬──┘
            对话/指令  │       │ 显式反馈(喜欢/跳过)
                       ▼       ▼
┌──────────────────────────────────────────────────┐
│                    写 入                          │
│                                                  │
│  extractFacts()  ──→ 12种情绪模式 正则匹配        │
│  extractFactsLLM()──→ DeepSeek 分析对话（每30条）  │
│  rememberLine()  ──→ memory.md Agent手写记忆      │
│  feedbackBoost() ──→ 喜欢↑跳过↓ 调整置信度         │
└──────────────────────┬───────────────────────────┘
                       ▼
┌──────────────────────────────────────────────────┐
│                    管 理                          │
│                                                  │
│  decayFacts()     ──→ 情绪指数/偏好线性衰减       │
│  consolidateFacts()──→ 去重合并相同事实           │
│  低置信度清除     ──→ confidence < 0.3 自动删除   │
│  上限控制         ──→ 最多100条，超出删最旧的     │
└──────────────────────┬───────────────────────────┘
                       ▼
┌──────────────────────────────────────────────────┐
│                    读 取                          │
│                                                  │
│  getEmotionalContext()   ──→ 当前情绪（1词）       │
│  getSessionTopic()       ──→ 当前话题+场景         │
│  recallMemory(query)     ──→ 按关键词搜索记忆      │
│  getRelevantFacts(topic) ──→ 按话题匹配            │
│  buildMemoryContext()    ──→ 分层构建LLM上下文     │
│    ① 长期记忆(memory.md)                          │
│    ② 用户画像(高置信度facts)                       │
│    ③ 听歌模式(统计分析)                            │
│    ④ 近期回顾(跨会话摘要)                          │
│    ⑤ 当前对话流                                   │
│  上下文上限: 1200字                                │
└──────────────────────────────────────────────────┘
```

### LLM 获取的上下文示例

```
## 当前感知
用户情绪: 疲惫
相关记忆: 用户深夜听歌；用户感到疲惫/困倦；喜欢连续听歌不停歇

## 长期记忆
- 用户深夜时段活跃 氛围感和摇滚交替
- 最近迷上了草东没有派对

## 用户画像
习惯: 用户深夜听歌；喜欢连续听歌不停歇
偏好: 喜欢听周杰伦；喜欢听草东
情绪: 用户感到疲惫/困倦；用户心情烦躁

## 听歌模式
时段: 晚上(50%)、下午(20%)
歌手: 周杰伦、草东、陈粒

## 近期回顾
7/13-7/15 — 操作: 漫游→继续→切歌 — 关键: 用户深夜听歌；偏好漫游
```

### 记忆生命周期

```
创建 ──→ 观察确认(↑置信度) ──→ 稳定存储 ──→ 不再观察(↓衰减) ──→ <0.3 删除
  │         ↑                    │
  │   每轮extractFacts           │  7天后开始衰减
  │   每次feedbackBoost          │  情绪3天半衰
  │   每30条LLM提取              │
  └──────────────────────────────┘
```

---

### 反馈信号映射

| 用户行为 | 记忆影响 |
|---------|---------|
| 点赞 ♡ | 该歌手偏好 +0.15 |
| 取消赞 | 该歌手偏好 -0.15 |
| 切歌 ⏭ | 当前歌手偏好 -0.15 |
| 说"喜欢X" | extractFacts 提取偏好事实(0.7) |
| 说"烦/累/emo" | extractFacts 提取情绪事实(0.6) |
| 完整听完 | 播放统计+1（songStats） |
| 说"漫游" | 习惯事实 +0.8 |

### API 清单

```js
// 写入
rememberLine(text)           // Agent写入memory.md
forgetLine(pattern)          // Agent删除memory.md条目
feedbackBoost(pattern, pos)  // 反馈驱动置信度调整

// 管理（自动触发）
decayFacts()                 // module load + maybeSummarize
consolidateFacts()           // maybeSummarize

// 读取
getMemorySnapshot()          // memory.md全文
recallMemory(query, n)       // 按关键词搜索
getRelevantFacts(topic, n)   // 按话题匹配
getTopFacts(n)               // 最高置信度N条
getEmotionalContext()        // 当前情绪（1词）
getSessionTopic()            // 当前话题+场景
buildMemoryContext()         // 分层LLM上下文
analyzePatterns()            // 播放模式统计
```
